欢迎使用！

需要配合MYSQL+PHP使用

zhfz文件夹是客户端：
    active是客户端活动管理
        index用于查找活动，active用于加入活动或跳转。sample文件夹内包含两个示例，但没有添加音乐和视频文件；doc里面是活动文件
    admin是设置：
        index用于主页面，bush是用于命令管理，app里面是命令管理的文件。所有命令见MYSQL文件夹
    loged是登录主框架
    metro文件夹是CSS、JS、FONTS的存储
    register是用于旧版学号查询
    search是通知界面
    about是关于
    index是登录
    logout是登出
root文件夹是管理界面
    active是活动管理
        index用于查找活动，active用于加入活动或跳转。doc是管理文件。注意0000活动必须存在，它作为添加活动
    admin是设置：
        index用于主页面，bush是用于命令管理，app里面是命令管理的文件。所有命令见MYSQL文件夹，admin是密码设置
    loged是登录主框架，包含权限检查
    push是推送：
        index用于主页面，push是后端。包含权限检查
    zyfz是学时
        index用于主页面，score用于添加学时、sqm是生成授权码、还用到了admin里面的批量学时添加
    about是关于
    index是登录
    logout是登出
